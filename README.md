# Multi-Modal AI System for Industrial Quality Assurance

## Overview

This repository documents my progress in the Summer of Code project focused on developing a Multi-Modal AI System for Industrial Quality Assurance.

The first phase of the project focused on building foundations in:

- Python Programming
- Machine Learning
- Deep Learning
- Convolutional Neural Networks
- Large Language Models
- Exploratory Data Analysis

The long-term objective is to develop a system capable of utilizing multiple modalities such as images, sensor information, and textual descriptions for automated quality inspection.
